Display accessibility statistics and reports directly in your WordPress admin dashboard. This plugin requires a paid subscription to the tidyDOM accessibility service.

== Description ==

[tidyDOM](https://tidydom.com) is a paid service that scans and continuously monitors your site for accessibility issues. This plugin provides access to accessibility reports directly in your administrative dashboard.

If you experience any issues with this plugin or need help setting it up, please contact us at support@tidydom.com.

[See Terms of Use](https://tidydom.com/terms).

== Installation ==

1. Upload `tidydom` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit `Settings` > `tidyDOM` in the administrative dashboard.
4. Enter the API Key you created for this site at tidDOM.com, and select the user roles that you would like to have access to the reports page.
5. Results from your latest accessibility scan will now be available at `Tools` > `Accessibility Report` in the administrative dashboard.

== Frequently Asked Questions ==

= Where do I find my API Key? =

* From your site dashboard on tidyDOM.com, go to the `Integrations` menu item and click "Add Integration".
* Select the `CMS Plugin` integration from the list.
* The API Key will be displayed on the next screen. Make note of it as you will not be able to view it again.

= What do I do if I've lost my API Key? =

You can generate a new API key from your site dashboard on tidyDOM.com.
* From your site dashboard on tidyDOM.com, go to the `Integrations` menu item, find the `CMS Plugin` integration and click `Configure`.
* Click the `Regenerate Key` button and confirm.
* The new API Key will be displayed and your old API Key will be disabled.
